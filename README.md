# Woy App

Haiti's social video platform.

## Deploy on Render

1. Push this folder to GitHub
2. Go to render.com → New → Static Site
3. Connect your GitHub repo
4. Set Publish Directory to `.`
5. Click Deploy

## Local

Just open `index.html` in your browser.
